# Proyecto Hola Mundo en PHP

## Nombre del proyecto
Aplicación Hola Mundo con PHP

## Descripción
Este proyecto consiste en una página web sencilla desarrollada en PHP que muestra en pantalla un mensaje de "Hola Mundo" junto con una imagen.  
El objetivo del proyecto es practicar la configuración de un servidor web en Ubuntu y ejecutar código PHP utilizando Apache.

## Prerrequisitos
Para ejecutar este proyecto es necesario contar con:

- Ubuntu Server
- Apache2 instalado
- PHP instalado
- Visual Studio Code
- Navegador web
- Git (para control de versiones)

## Pasos para ejecutar el proyecto

1. Instalar Apache2 y PHP en el servidor Ubuntu.
2. Copiar la carpeta del proyecto dentro de la ruta: /var/www/html
3. Iniciar el servidor Apache si no está activo.
4. Abrir un navegador web y escribir la siguiente dirección: http://127.0.0.1:8080/proyecto_php/index.php
5. Al abrir la página se mostrará el mensaje **Hola Mundo** junto con la imagen incluida en el proyecto.
